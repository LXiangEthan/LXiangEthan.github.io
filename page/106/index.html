多态中父类的虚函数没有一点用处，就可以把父类的虚函数写成纯虚函数

当类中有了纯虚函数，这个类就变成抽象类
抽象类无法实例化对象（栈和堆都不能实例化）
子类如果要实例化对象必须要重写纯虚函数

多态实现条件：
1、必须有继承关系
2、子类重写父类中的虚函数，子类中对于这个函数是否为虚函数是可有可不有的
父类的指针或引用指向子类对象


C++中的父类中的成员变量，在进行多态后，指向的仍然是父类的成员变量

我们发现有虚函数的类里面，类的大小是4个字节（整型，浮点型，指针都是四个字节），虚函数所在类里面就是多出来一个指针（vfptr/virtual function pointer）,虚函数（表）指针，指向vftable（虚函数表，virtual function table），表内部记录虚函数地址。
子类中继承的是虚函数指针，指向子类里面虚函数表，当子类中重写了父类的虚函数，子类中的虚函数表会替换成子类的虚函数地址，当父类指针或引用指向子类对象的时候，发生多态
