foreach

foreach是遍历操作
会自动把数组遍历一遍
调用的回调函数里面传的第一个参数就是数组每个的内容，第二个参数是数组里面每个的索引值，第三个参数是原数组

map

map映射,map会把数组里面每一个值都映射入回调函数，每一项都会被返回值取代,回调函数第一个值就是数组的每一项

filter

filter过滤,filter只有两个返回值，true和false，如果返回true就满足条件，映射出来，不满足条件就不映射出来,传入的第一个参数仍然是每个数组元素

every

every每一个,返回值只有布尔值，但是every判断的是每一个数组元素，如果全部满足就返回true，只要有一个不满足就返回false

some

some只要有一个满足条件，就是true

find

find找到满足条件的元素，find和filter是一样的满足条件就映射，但是find只能找到第一项，filter能找到所有的

reduce

reduce叠加,reduce有两个参数一个是回调函数，一个是就是初始值就是prev的初始值，reduce的回调函数还有两个参数，这里的两个参数一个是prev，这是在遍历数组过程中上一次的return结果，item才是当前的每个数组元素

concat

concat实现数组之间的拼接，返回一个拼接后的新数组，阔号里面可以写多个值，这些值中可以只传一个元素
concat也可以不传参数,这种方式就是实现数组的复制操作

join

在我们把数组打印在文档中是由逗号分隔的字符串
join可以实现数组转换成字符串，然后修改逗号分隔为其他符号
join数组转字符串

slice

slice截取数组（包前不包后）
1、传两个参数：slice(开始索引，结束索引)，从开始索引截取到结束索引，包前不包后，包括开始索引不包括结束索引
2、传一个参数：slice(开始索引)，从开始索引一直截取到最后
3、不传参数：相当于复制了一份原数组（这也是一种数组赋值的方法）
4、传负数：arr1.slice(2,-1)，负值就相当于从数组最后面数出来的参数，最后一个元素是-1，包前不包后

indexOf

indexOf 返回查找的索引值
返回-1就是没有找到当前的元素，找到了会返回查找元素的索引值
但是这种方法只能查到遇到的第一个，如果有重名的找不到后面的元素
arr.indexOf("liangxiang",2)后面的2是，从索引为2的开始查找

lastIndexOf

lastIndexOf和indexOf的区别只有lastIndexOf是从数组最后面查找，同样找到第一个就不会往后面查找了
