瀑布流效果


column-count:5;
//分成列的个数
column-gap:30px;
//调节列之间间距
column-rule:2px solid red;
//列之间的边框
column-fill:balance;
//列高度统一
column-fill:auto;
//每个父盒子占满
column-width:500px;
//调整列宽

加给小项目，让这个项目横跨所有列
column-span:all;
