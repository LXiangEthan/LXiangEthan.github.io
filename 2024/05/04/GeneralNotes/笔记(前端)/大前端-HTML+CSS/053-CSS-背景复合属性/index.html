  1. 用空格隔开
  2. 顺序可以换
  3. 可以只取一个值， 放在后面能覆盖前面的值
    background:url(img/1.jpg) no-repeat center fixed yellow;
    background-size需要单独设置，不能用于复合属性中

这里有一个点，background的复合属性设置的方法会把前面所有除background-size外有关背景的属性设置全部覆盖

像这样黄色的背景颜色就会被覆盖成没有设置
