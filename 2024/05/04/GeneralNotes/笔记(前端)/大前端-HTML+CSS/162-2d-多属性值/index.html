缩放，旋转，平移写在一个属性中
transform: translateX(400px) scale(0.5) rotate(45deg);

如果对盒子先缩放，再位移，会导致位移也会缩小，要先写位移再写缩放

如果先旋转，再位移，就会导致坐标轴旋转，向旋转方向位移
所以要先位移再旋转和缩放,而且写在同一个transform里面
