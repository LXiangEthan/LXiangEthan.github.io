1、兄弟关系，两个盒子垂直外边距与水平外边距问题
2、父子关系，给予子盒子外边距，但作用于父盒子身上怎么解决

1）给兄弟关系的两个盒子都设置外边距，垂直方向上会取两个盒子的外边距中的最大值，不会相加，水平方向上，则不会取最大值，两个外边距就会相加

2）如果子盒子和父盒子之间没有边距。子盒子设置外边距那么子盒子的外边距就会转移到父盒子上
解决方法：
1、给父盒子加上内边距，（效果不好）
2、给父盒子设置边框，这样父盒子和子盒子之间就会有边框的距离，就能解决问题
3、让子盒子加上浮动或者给父盒子加浮动就能解决问题
4、父盒子设置BFC，overflow:hidden;不会受到外界打扰
设置颜色可以使用transparent实现透明属性
