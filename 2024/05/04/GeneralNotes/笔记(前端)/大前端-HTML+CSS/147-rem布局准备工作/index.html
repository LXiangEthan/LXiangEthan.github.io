js动态设置html上的字体大小（CSS宽度，并非物理宽度）：

物理宽度还要再除以一个2，这样设置其他盒子的宽度就是设计稿的物理宽度
fontSize = 当前设备的css布局宽度/CSS分辨率*基准fontSize
这种方法还有点小问题，其他盒子会再继承html的字体，可以给body再设置一个字体16px

借助工具计算rem
vscode—px to rem & rpx
用750px后面再加上一个‘-’，回车就自动转换成rem了
使用F1，搜索cssrem，把全部页面都转换成rem
但是整个插件就是基于16px来转换的，可以在文件-首选项-搜索px-Cssrem:Root Font Size修改
