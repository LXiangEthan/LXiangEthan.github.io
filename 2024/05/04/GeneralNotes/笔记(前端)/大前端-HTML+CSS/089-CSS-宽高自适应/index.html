宽度不写，就是宽度自适应width:auto;，宽度自动占满
width:auto;和width:100%;不同，如果设置内边距，那么盒子的宽度就是100%+padding，而不是自适应
高度自适应是有子元素撑开的高度，默认是没有高度的，不设置高度那么就会由子元素自动撑开高度

min-height:–px;
min-width:–px;
如果内容过小，使用最小宽/高度可以使页面更美观,达到自适应
max-height:–px;
max-width:–px;
如果内容过多，设置最大宽/高度可以达到自适应，防止页面过大
