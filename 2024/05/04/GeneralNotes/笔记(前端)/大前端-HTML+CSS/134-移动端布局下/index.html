所有移动端的前提是这个meta标签，没有这个标签，所有的移动端属性都会失效

移动端会自动带一个水平竖直的滚动条，这时用于放大缩小用的，以前没有专门的移动端网页的时候，可以利用这项功能用手机打开pc端网页，缩放屏幕，可以看清pc端网页上面的字


使用这个标签可以让布局视口变成视觉视口的大小，这就是理想视口，让网页的宽可以等于手机的宽，这样就可以单独写一个移动端的布局（简而言之就是加上这个标签可以让手机浏览器的网页变成理想视口）

禁用移动端缩放功能

加上user-scalable=no就可以禁用缩放功能
