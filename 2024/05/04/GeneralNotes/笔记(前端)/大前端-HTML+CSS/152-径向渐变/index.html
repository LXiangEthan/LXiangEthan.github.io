中心点向四周渐变

可以直接写

默认从中心向四周扩散

调整显示大小

0%-10%红变黄，10%-30%黄变蓝
如果设置成相同的变换区间就不会渐变（太极就可以利用这个写出来）

设置径向渐变的形状

要求形状就是一个圆
ellipse就是椭圆
如果渐变的是一个正方形，那么椭圆和圆是一样的，但是在长方形中不一样

改变中心位置

水平方向60%，竖直方向40%为中心点
closest-side指定渐变到最近的边渐变就停止
farthest-side最远边，closest-corner最近角，farthest-corner最远角

background:radial-gradient(center,shape,size,start-color,····,last-color);这个属性兼容性不太好chrome需要在前面加上-webkit-,firefox需要在前面加上-moz-
