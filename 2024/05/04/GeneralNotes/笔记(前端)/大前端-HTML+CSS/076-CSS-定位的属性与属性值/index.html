static就是静态默认定位
absolute绝对定位–按照窗口位置定位
绝对定位生效条件是有父元素而且父元素要有定位属性
relative–相对定位，按照元素的初始位置移动
fix–固定定位，固定在某个位置上
sticky–粘性定位，可以粘在某处
配合top,right,left,bottom使用
