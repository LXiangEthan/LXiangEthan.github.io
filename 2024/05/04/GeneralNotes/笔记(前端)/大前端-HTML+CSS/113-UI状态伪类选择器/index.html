:enabled可用的表单可以被选中
:disabled可用的表单可以被选中
:focus可用的表单可以被选中
:checked表单中选中的的部分会受到这个选择器影响
checkbox的多选表单在这里不受上述UI状态伪类选择器选中,因为它受到默认样式的管控
但是可以把它的默认样式去掉appearance:none;就可以受到上述选择器的影响
:selection鼠标选中字体可以修改背景颜色和字体颜色
