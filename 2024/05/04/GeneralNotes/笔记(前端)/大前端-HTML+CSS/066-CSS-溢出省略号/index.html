white-space
normal—默认撑开盒子换行
nowrap—不折行
pre标签用来保留空格、tab、回车等部分
white-space:pre;就和pre标签达到同样效果
pre-wrap–显示空格，tab，回车，换行;
pre-line–显示回车，不显示空格，换行

显示省略号：
text-overflow:ellipsis;
溢出的部分变成省略号
