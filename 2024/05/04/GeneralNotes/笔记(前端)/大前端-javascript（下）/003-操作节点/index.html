box.appendChild 在元素最后插入节点

box.insertBefore(obi,near) 在元素near的前面插入节点

box.removeChild(near) 删除节点near
box.remove() 删除节点box及其后代

box.replaceChild(新的节点，老的节点) 替换节点

var onCloneBox = box.cloneNode() 克隆box节点（不克隆后代）
如果不写参数，默认是false，不克隆后代
var onCloneBox = box.cloneNode(true) 克隆box节点（克隆后代）
克隆会把id一起复制，可以再指定一个id

document.createElement 创建元素节点

document.createTextNode 创建文本节点

document.createComment 创建注释节点

document.createDocumentFragment 创建文档片段节点

document.body.insertBefore(要插入的节点，插入的位置); 在指定位置前插入节点
