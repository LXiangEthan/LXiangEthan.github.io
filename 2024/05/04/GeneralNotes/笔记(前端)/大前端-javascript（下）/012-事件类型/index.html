鼠标事件

click 点击

dblclick 双击

contextmenu 右键单击

mousedown 鼠标按下
mousemove 鼠标移动
mouseup 鼠标抬起

mousedown和click的区别是：
click如果按下去可以反悔，把鼠标移开在抬起鼠标，这样就不执行
但是mousedown不能反悔，鼠标按下去的一瞬间，就执行了
//————————-
mousemove
鼠标在btn1上划过去就会执行函数
//————————-
mousedown
鼠标在btn2上面抬起后就会执行函数

mouseover mouseout 鼠标移入移出

mouseenter mouseleave 鼠标移入移出

mouseenter mouseleave和mouseover mouseout 的区别是mouseover mouseout 会在盒子的孩子盒子也加上移入移出事件，而mouseenter mouseleave 不会加在孩子身上

keydown键盘按下事件
keyup


表单事件
focus 获取焦点事件
blur 失去焦点事件

change 获取焦点 失去焦点对比里面内容不一样才会触发

input 内容不一样就会触发

submit 表单提交触发
reset 表单重置触发
如果想让这两个属性触发就必须要使用form标签
input的提交按键就会自动跳转页面，需要阻止默认跳转行为，在函数中使用return false

触摸事件（只针对移动端）
touchstart 开始摸
touchmove 移动过程中
touchend 移动结束

左滑右滑画圈都是有这几个事件模拟出来的
box.ontouchcanael事件（在触摸网页中被电话打断会触发）
