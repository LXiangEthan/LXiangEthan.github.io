认识Nodejs
Node.js是一个javascript运行环境。它让javascript可以开发后端程序，实现几乎其他后端语言实现的所有功能，可以与PHP、Java、Python、.NET、Ruby等后端语言平起平坐。Nodejs是基于V8引擎，V8是Google发布的开源JavaScript引擎，本身就是用于Chrome浏览器的js解释部分，但是Ryan Dahl 这哥们，鬼才般的，把这个V8搬到了服务器上，用于做服务器的软件。

NodeJs能实现超高的高并发，性能方面杠杠的

01 nodejs的特性
  • Nodejs语法完全是js语法，只要你懂js基础就可以学会Nodejs后端开发
  • NodeJs超强的高并发能力,实现高性能服务器
  • 开发周期短、开发成本低、学习成本低
02 使用 Node.js 需要了解多少 JavaScript

http://nodejs.cn/learn/how-much-javascript-do-you-need-to-know-to-use-nodejs

Node只有V8引擎，没有Blink内核，所以只能用js语法
还有一些中间层libuv可以实现一些浏览器不能实现的功能
Nodejs解析的js代码（没有浏览器安全级别的限制），提供很多系统级别的API
比如：

  • 文件的读写 (File System)

    • 进程的管理 (Process)

  • 网络通信 (HTTP/HTTPS)

    Node没有了浏览器单线程的限制，可以创建多个进程
    Nodejs和js最大区别就是Node没有了浏览器的限制
