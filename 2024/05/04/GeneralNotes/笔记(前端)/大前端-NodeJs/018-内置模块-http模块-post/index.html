小米有品这个网站的数据必须要post请求才能拿到
浏览器地址中直接访问就是get请求
POST请求需要给后端带上一些数据，在payload中找
POST请求需要使用request模块，因为没有https.post^-^
而且POST请求不能只有域名地址，，还需要一些配置
比如域名，端口号，路径，请求是post
配置完了还不够post请求的数据是有讲究的：有的要求是application/json,有的要求application/x-www-form-urlencoded,前后端必须要一致
那么我们怎么用post发数据呢，https.request请求会返回一个对象这个对象里面有一个write方法可以发数据
在小米有品中需要在post里面给后端一个数据
[
{},
{
“baseParam”: {
“ypClient”: 1
}
}
]
这样才能获取到数据
post同样也是流的方式传输
这里需要把数据转成json格式post给小米有品
#这里要注意一下，域名前面就不要加协议了
