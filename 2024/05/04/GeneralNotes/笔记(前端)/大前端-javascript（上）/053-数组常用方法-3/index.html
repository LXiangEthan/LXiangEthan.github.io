foreach

foreach是遍历操作
会自动把数组遍历一遍
调用的回调函数里面传的第一个参数就是数组每个的内容，第二个参数是数组里面每个的索引值，第三个参数是原数组

map

map映射,map会把数组里面每一个值都映射入回调函数，每一项都会被返回值取代,回调函数第一个值就是数组的每一项

filter

filter过滤,filter只有两个返回值，true和false，如果返回true就满足条件，映射出来，不满足条件就不映射出来,传入的第一个参数仍然是每个数组元素

every

every每一个,返回值只有布尔值，但是every判断的是每一个数组元素，如果全部满足就返回true，只要有一个不满足就返回false

some

some只要有一个满足条件，就是true

find

find找到满足条件的元素，find和filter是一样的满足条件就映射，但是find只能找到第一项，filter能找到所有的

reduce

reduce叠加,reduce有两个参数一个是回调函数，一个是就是初始值就是prev的初始值，reduce的回调函数还有两个参数，这里的两个参数一个是prev，这是在遍历数组过程中上一次的return结果，item才是当前的每个数组元素
