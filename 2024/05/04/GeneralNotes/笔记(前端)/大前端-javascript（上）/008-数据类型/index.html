1、基本类型
1)Number 1（整数）、12.5（浮点数）、2e3（科学计数法2*10^3）、0x99（16进制）、077（8进制）、0b11（2进制）
在控制台中数字是有颜色的

2)String “1000” ‘hello’
在控制台中字符串是没有颜色的,单引号和双引号在js里面是一样的

3)Boolean true false
只有两个值 true false,代表真和假

4)Undefined 声明了未赋值的变量
用于以后不知道需要赋什么值的变量，先放在这，不定义

5)Null 空对象
null用于知道未来这个值将会赋值对象类型

这一节不介绍
