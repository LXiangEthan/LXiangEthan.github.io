为什么set不能重复而multiset可以重复
因为在set里面插入时有一个叫对组的pair返回，包括了插入的位置和是否插入成功，在插入过程中，如果发生重复就会在插入之前就返回插入失败，就导致无法插入，而multiset可以插入重复的值因为它的返回值不会包括对重复的检测不会返回插入成功或失败
