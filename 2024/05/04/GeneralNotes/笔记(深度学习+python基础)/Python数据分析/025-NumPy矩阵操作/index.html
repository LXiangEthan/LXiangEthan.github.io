线性代数常用方法

n = np.array([[1, 2, 3], [4, 8, 6], [7, 8, 9]])  
  
# 线性代数方法,方法存放在np.linalg这个对象中  
np.linalg.inv(n)  # 矩阵求逆，这里的条件是矩阵要可逆，如果矩阵不可逆，那么就不能求  
np.round(np.linalg.det(n))  # 求行列式,round四舍五入  
np.linalg.matrix_rank(n)  # 求出来矩阵的秩（满秩矩阵，秩等于行数）（奇异矩阵，秩小于行数）
