ndarray是NumPy中表示数组的重要数据类型

使用np.array()创建
这个方法和我们的list()，tuple()等方法是一样的，我们直接把数组，在Python中应该叫列表直接放进去就可以直接转换成ndarray数组了
这个地方有一个点，那就是array()方法只能得到一维数组，否则报错

如果出现了不同数据类型，那么按照优先级最高的来统一

利用np.array()得到ndarray数组

import numpy as np  
  
l = [1, 2, 3, 4, 5, 6, 7, 8, 9]  
matrix = np.array(l)  
# 得到ndarray数据类型  
matrix.shape  
# 这个可以获取到元组，如果只有一个数字，那么这个ndarray数组就是一维的，单纯的列表是没有这个属性的  

(9,)  

m = np.array([1, 2, 3, 4, 5, 6, 'Hello'])  
m  

array(['1', '2', '3', '4', '5', '6', 'Hello'], dtype='<U11')

利用np的routine()来创建

利用np.ones(shape, dtype=None, order=’C’, *, like=None):创建

shape形状,是一个元组 dtype元素类型 order[‘C’,’F’],c表示以列为主，f表示以行为主 

s = np.ones(shape=(9, 9, 1), dtype=np.int8, order='C')  
# 默认往数组中添加的数据就是1  
# print(s)

借助np.zeros(shape, dtype=None, order=’C’, *args, **kwargs):创建

ss = np.zeros(shape=(9, 9, 1), dtype=np.int8, order='C')  
# 默认在数组中添加的数据就是0  
# print(ss)

sss = np.full(shape=(9, 9), fill_value=9, order='C')

利用np.linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=None,axis=0):创建

创建等差数列

  • start开始值
  • stop结束值
  • num等差数列个数
  • endpoint是否包含结束值
  • retstep是否返回等差值（是否返回步长）
  • dtype数据类型
equal_list = np.linspace(1, 50, 20, True, True, np.int8)  
print(equal_list)  
# 会得到一个元组，这个元组中包含了等差数列的数列，数据类型，步长
