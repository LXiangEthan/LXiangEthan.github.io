概述

  • 爬虫是批量化自动化获取即有数据
    • 批量化
    • 自动化
    • 既有数据
  • 通常来说都是获取即有数据
    • 都是获取既有数据
  • 也有特殊的
    • 批量注册一些账号
    • 批量的去领取优惠券
    • 批量下载商品
    • 自动做任务（签到）

实际应用

  • 在企业中
    • 应用最多的
      • 做竞品调研数据采集
    • 办公自动化
      • 自动从公司的后台的地方获取数据进行处理
  • 当个人服务
    • 爬取网络免费资源
    • 抢票

分类

  • 四大类
  • 聚焦爬虫
    • 完成某一特定数据的采集
    • 百分之九十的爬虫
  • 通用爬虫
    • 什么内容都爬取，都存储下来
    • 搜索引擎
  • 增量爬虫
    • 既可以实现聚焦爬虫
    • 也可以是通用爬虫
    • 当内容发生变化的时候，它可以增量的获取内容
  • 暗网爬虫
    • 深网爬虫
    • 很少见
    • 暗网/深网，本身就是见不得光的地方
    • 暗网爬虫既可以是聚焦爬虫，也可以是通用爬虫，当然也可以是增量爬虫

爬虫合法吗？

机器人协议

  • robots.txt
    • https://www.baidu.com/robots.txt 百度的机器人协议
    • https://www.taobao.com/robots.txt 淘宝机器人协议，淘宝和百度好像有点仇
    • 通常协议中会标明哪些不让爬，但是像京东这样顶级的购物网站都没有写机器人协议

君子协议

  • 为标注是否可以爬去
  • 历史上有哪些爬虫工程师吃上了官家饭
    • 有一家公司被连窝端了
      • 工程师写了一个爬虫，采集另外一家公司的网站数据，丝毫没有管网站的死活
    • 有一个研究生写了一个爬虫给女朋友抢HPV疫苗，给别人抢，收费

爬虫的注意事项

  • 给服务器留活路
    • 爬虫开启后，使用其他设备去访问爬虫网站
  • 脱离敏感数据
  • 不要公开收费
  • 不要爬国家这种公共资源的网站
