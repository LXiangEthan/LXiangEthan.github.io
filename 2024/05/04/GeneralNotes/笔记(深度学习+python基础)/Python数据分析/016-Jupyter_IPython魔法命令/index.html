%run


%run可以直接运行我们的python文件，但是它这个不是调用的运行，而是直接把文件中的代码放在我们的代码块中使用，这样下面的代码块就可以使用我们文件中定义的函数或方法

%time

统计运行的时间
%time 运行代码，只运行一次，测出时间长短

  • %time一般用于运行时间较长的代码

%timeit

  • 对于短时间的代码，我们可以使用%timeit来测出时间，这个可以测出纳秒值，这条魔法命令会反复的执行得到运行时间
  • 这边有一个坑一定要注意，对于一般的函数运行，%time命令就是足够了的,%timeit是针对于那些运行时间极短的程序，我们这边遇到了如果使用%timeit方法运行一个函数会报错运行速率的问题，当我们修改了配置文件，程序可以运行，但是运行过程中CPU占用过多导致崩溃，因为我们的%timeit是计算纳秒级别的时间，用于赋值或者加减法等可以，对于一个程序运行太吃内存了
  • 对于%timeit命令我们可以借助传参数来实现对时间的计算 上面表示-r标识计算多少轮，-n标识每轮计算多少次

%%timeit

  • 两个百分号表示我们可以一次运行多行代码，获取到的是每行代码的平均运行时间
# with open("demo.py", 'w', encoding='utf-8') as f:  
#     f.write("print('Hello Jupyter')")  
%run demo.py  

%timeit -r 3 -n 1000000 a=11  

9.7 ns ± 0.833 ns per loop (mean ± std. dev. of 3 runs, 1,000,000 loops each)

运行3轮，每轮计算11次 

%%timeit  
a = 11  
b = a  
b += 11  

26.4 ns ± 0.791 ns per loop (mean ± std. dev. of 7 runs, 10,000,000 loops each)

计算三行代码每次运行时间
