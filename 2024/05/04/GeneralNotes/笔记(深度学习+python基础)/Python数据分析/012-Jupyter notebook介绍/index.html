启动jupyter notebook

在conda环境或者虚拟环境中直接输入jupyter notebook
然后浏览器就会自动打开浏览器，如果浏览器没有自动打开，就拷贝下来终端上打印的地址，使用浏览器打开

或者还有一种打开方式，在搜索中找到jupyter notebook直接点击打开
