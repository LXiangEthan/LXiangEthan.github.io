核心

  • 常用API没有必要刻意的去记住
  • 这边我们使用驱动就相当于控制力chrome浏览器，让浏览器去为我们做事，但是记住这是我们自动化程序控制的没事，不要点击
from selenium import webdriver  
driver = webdriver.Chrome()

创建chrome的驱动 

driver.get("https://www.xinpianchang.com")

控制浏览器想该地址发送get请求 

driver.get("http://www.jd.com")

再次打开一个页面 

driver.back()

回退页面 

driver.forward()

前进页面 

elements = driver.find_element('id',"hello")

查找当前页面的元素 

# elements.send_keys("爬虫")  
elements.click()

向输入框中发送值 

elements_2 = driver.find_element("id","rounded-lg")

查找第二个元素 

elements_2.click()

点击元素 

the_page = driver.page_source

获取页面内容 

driver.close()

关闭驱动
