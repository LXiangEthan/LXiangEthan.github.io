当目标文件中既有文件又有文件夹时，我们使用listdir()方法就只能获得第一层子文件或文件夹了，而子文件夹中的内容便获取不到了。
这时候我们需要用到os.walk()方法：传入目标路径即可。该方法可以递归的找出目表路径下的所有文件，无论他藏的有多深~
