如果导入的功能重复名称，那么后面导入的就会把前面导入的功能覆盖掉，如果我们都想要使用，可以借助别名



在我们python模块中每个模块对象默认有一个_name_的成员变量，一旦我们运行该模块，那么编译器就会赋值__name__为”__main__“,然后我们可以借助这个特性来便捷的调用模块内的功能，防止其他模块调用我们功能的时候我们自己的调用也被执行了
