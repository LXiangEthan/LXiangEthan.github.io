在mybatis官网上有一个专门的章节讲解动态sql
https://mybatis.net.cn/dynamic-sql.html

使用if标签可以实现动态sql，if标签使用来完成条件判断的，如果满足条件那么if标签里面的内容就会显示，否则就没有，if标签里面写test属性，就是写条件的地方

但是这里还有问题，不知道大家有没有发现，那就是如果我们第一个name参数如果没有怎么办?
我们会发现，程序报错了，不难看出来，因为and的存在，如果没有第一个name参数，那么where后面就会直接跟着一个and，导致sql语句报错

解决方式：
笨办法，在where后面加上一个恒等式1=1，代表一定满足，这样即使后面多出来一个and程序也不会报错

聪明办法：使用where标签

使用where标签可以自动删除语句前面的and字段，而且也不用我们自己写where

单条件的动态条件查询：

choose == switch
when == case
otherwise == default

Test.java

映射文件

如果我们在choose中并没有匹配上的我们可以利用otherwise来默认选中，或者利用where标签，where标签就是会根据我们后面的语句，如果后面语句不存在那么where就会删除
