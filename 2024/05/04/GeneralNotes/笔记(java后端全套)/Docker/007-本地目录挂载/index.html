通过docker inspect mysql就能查看容器的挂在信息

发现mysql有一个匿名卷
这个匿名卷是用于mysql数据存储的挂载目录

mysql为什么有这个匿名目录是因为，mysql在运行过程中会不断的产生数据，如果不把它数据存储的文件夹挂在出来，那么容器内的存储占用会越来越高，这是不利的

这里我解释一下数据卷挂载，挂载就是把容器里面的数据放到外面宿主机中，数据存储在宿主机中，但是容器里面使用这个数据
