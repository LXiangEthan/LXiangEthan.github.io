docker pull 下载拉去镜像
docker images 查看本地镜像
docker rmi 删除本地镜像
docker build 构建自己的打包镜像
docker save 把本地镜像保存压缩到本地
docker load 加载压缩后的镜像
docker push 推到镜像仓库

docker run 为镜像创建对应容器并运行，如果镜像不存在会先拉取
docker stop 停止运行容器
docker start 把停止的容器启动起来

docker ps 查看当前容器的运行状态（不加参数只查看运行中的容器）
docker rm 删除本地容器
docker logs 查看日志
docker exec 进入容器内部
如果要全部的命令请查看官方文档
