java中对于引用数据类型和其他的集合集合中存储的都是地址
而对于基本数据类型，集合存放的是真正的数据
另外我还要说的是java的垃圾回收机制就是会自动回收没有被指向的引用数据类型，所以在作用域中因为有指向的存在，所以引用数据类型不会被回收，这就是为什么我们在作用域内把引用数据类型添加到集合中，但是数据却没有被JVM当成垃圾清除
