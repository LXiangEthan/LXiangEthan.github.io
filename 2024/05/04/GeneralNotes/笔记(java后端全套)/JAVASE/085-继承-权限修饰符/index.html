private
protected
public
缺省
java有四个权限修饰符
但是protected和缺省很少用到

一个.java文件只能有一个public类，代表这个文件的入口，而且如果引入包，也只能引入带public的类，其他类无法单独调用
而且protected只能被子类调用，而不是子类对象调用
总结一下：
private:除了自己的类都不能访问
缺省：相当于public，但是只在一个包中生效
protected：除非继承否则不能访问
public:都能访问
