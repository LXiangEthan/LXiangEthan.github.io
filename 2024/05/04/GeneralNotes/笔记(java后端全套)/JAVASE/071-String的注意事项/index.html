java为了节约内存设计出来””直接赋值的String放在常量池里面




String类型就是常量，+拼接后得到的String不是直接基于原String修改，而直接在堆区创建新对象。只有通过”string”直接赋值的才会放在常量池
如果“str”+”str”直接相加，编译器会优化直接拼接后在赋值
