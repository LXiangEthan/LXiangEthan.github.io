因为Callable是一个泛型接口，所以我需要在这里解释一下，关于泛型类或者接口，泛型只能被泛型继承或实现，如果指定了数据类型，就可以被普通类继承（实现）

这边我要说一下，使用sleep方法的话需要处理它可能会出现的中断异常，但是因为我们重写的run方法在Thread类中并不能抛出异常，所以就需要我们自己在run方法内部处理异常了
重写方法也要也要遵循抛出异常的一致性
