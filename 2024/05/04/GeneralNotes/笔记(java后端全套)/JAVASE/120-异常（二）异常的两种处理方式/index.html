运行式异常会jvm会自动一层一层的往上一级抛出找到捕获异常的程序，可以不用手动抛出
但是编译式异常不会自动抛出，只要出现编译式异常如果不抛出，会无法编译
我们在写程序的过程中最好在我们自己的程序的最外层捕获异常
try catch可以使用多个catch捕获多个异常，throws也可以抛出多个同级类的异常
