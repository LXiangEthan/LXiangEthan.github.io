枚举类的构造器私有，不允许构造对象
枚举类的第一行都是常量，记住是枚举类的对象
枚举类中一些额外的api:

如果在枚举类型中出现了抽象方法，那么创建对象就需要重写这个抽象方法,这个创建枚举对象的方法还可以调用有参构造函数

枚举可以实现单例设计模式，只设置一个对象

枚举常见的应用场景：

枚举可以控制只有一定类数，而且在switch中枚举的类名直接删除了，编译器会自动识别这个枚举类型对象
