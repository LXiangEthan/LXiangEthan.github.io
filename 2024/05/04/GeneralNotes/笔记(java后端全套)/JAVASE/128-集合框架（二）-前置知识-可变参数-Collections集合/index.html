认识java中的可变参数可以顺便带着js中的arguments来认识下

java中的可变参数默认就是传入了一个数组，因此在可变参数就是一个数组，可以使用length方法和Array.toString方法

常见应用场景就是用在构造器上灵活使用参数的，但是java里面在一个方法内不允许定义多个可变参数的，只能定义一个

可变参数必须要放到形参列表的后面，不然就会出现矛盾，形参全部被可变参数占用了
