为什么会有这个问题？
文件字符输出流为了防止多次调用系统操作，当我们从内存的获取到数据，不会立马把数据写到文件中去，首先它会把数据放到一个缓冲区里面，最后把缓冲区的数据全部写入文件，这样优化了性能
不刷新或者关闭流，数据就不会从缓冲区写入文件
flush()刷新流,关闭流包含了刷新操作，关闭流之后不能继续使用，刷新流之后还能继续使用，缓冲区存放溢出会自动刷新
