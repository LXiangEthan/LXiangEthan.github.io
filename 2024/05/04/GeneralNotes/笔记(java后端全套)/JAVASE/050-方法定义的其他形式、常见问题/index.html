1、==方法再类中的位置前后无所谓，但是一个方法不能定义在另一个方法内部==
2、==方法的返回值写void，方法内不能使用return返回数据，如果方法的返回值类型写了具体类型，方法内部就必须使用return返回相应类型的数据==
3、==return语句的下面不能编写代码，属于无效的代码，执行不到这==
4、==方法不调用就不会执行，调用方法时传给方法的数据必须严格的匹配方法的参数情况==
5、==调用有返回值的方法，有三种方法，1、可以定义变量接受结果，2、或者直接输出调用，3、甚至直接调用==
6、==调用无返回值的方法，只有一种方法：1、只能直接调用==
