我们在浏览器中如果不输入端口号那么就是默认使用80端口号，这就是为什么的登录其他网站不需要输入端口号的原因，所以我们可以设置tomcat的默认端口号为80，那么我们就可以直接通过localhost进入tomcat了

我们把文件放到webapps文件夹下之后，就可以直接使用地址栏上的地址来访问我们的文件了

我们直接使用文件访问是比较慢的，我们可以把我们的web项目打包成一个war包，然后放到tomcat的webapps下面，tomcat就会自动帮我们解析和解压缩
