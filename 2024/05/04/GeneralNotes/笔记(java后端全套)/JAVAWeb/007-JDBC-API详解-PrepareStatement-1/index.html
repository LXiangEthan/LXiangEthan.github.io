PrepareStatement继承了Statement

我们把sql语句写入我们的登录框中，那么程序会认为我们的sql脚本是sql语句，这样就实现了对服务器的攻击
下面我仿照了老师成功实现了SQL注入,由此告诉我们sql注入很可怕，一定要预防

我们也可以借助其他方法来预防sql注入，但是有了prepareStatement，防止更好

我们将来一定不能让sql注入有机可乘
